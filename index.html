1.1.2 2019-10-23 星期三

  1. Org-Mode 表格基本使用

    [2019-10-23 周三 23:04]

    1. Org-Mode 代码
      #+name: test01
|     |  a |  b |  c |
|-----+----+----+----|
|     |  1 |  2 |  3 |
|     |  2 |  3 |  4 |
|     |  3 |  4 |  5 |
|     |  4 |  5 |  6 |
| sum | 10 | 14 | 18 |
#+tblfm: @6$2=vsum(@2$2..@5$2)
#+tblfm: @6$3=vsum(@2$3..@5$3)
#+tblfm: @6$4=vsum(@2$4..@5$4)


|-----+----+-----|
|   a |  c |   d |
|-----+----+-----|
| 123 | 45 | 258 |
|  18 |    |     |
|-----+----+-----|
#+tblfm: @3$1=remote(test01,@6$4)


注解

@ 表示行
$ 表示列
#+name: test01 表名称为test01
#+tblfm: @6$2=vsum(@2$2..@5$2)  求值公式表示（c-c c-c 表示求值）
vsum() 求和
@6$2=vsum(@2$2..@5$2)  第六行第二列等于第二行第二列到第五行第二列的和

remote(test01,@6$4) 调用test01表的第六行第四列的值
    2. 执行结果
        a b c
        1 2 3
        2 3 4
        3 4 5
        4 5 6
      sum 10 14 18

      第二张表

      a c d
      123 45 258
      18

3 Docker镜像构建与服务部署

使用前注意事项：

  1. 部署的服务前请先修改文件里面测试用的密码和端口
  2. 用于生产环境时请先查阅一遍文件，确认无误后再部署

3.1 gitbook

git clone https://github.com/luxiaoyou/Dockerfile.git
cd gitbook
docker-compose -d up

3.2 jenkins lts

mkdir -p /var/jenkins_home
git clone https://github.com/luxiaoyou/Dockerfile.git
cd jenkins
docker-compose -d up

浏览器访问

  • http://localhost:8080
  • http://<服务器ip>:8080

3.3 nodeppt

git clone https://github.com/luxiaoyou/Dockerfile.git
cd nodeppt
docker build -t nodeppt:8-alpine .

3.4 redmine

git clone https://github.com/luxiaoyou/Dockerfile.git
cd redmine
docker-compose up -d

redmine:

  • 浏览 localhost:8080 (部署好可能要等几分钟才能访问，或者你自动刷新，直到能访问)
  • 用户：admin 密码：admin 首次登录登录修改密码用
